Tooling: Copilot CLI & Instruction Files

Copilot CLI

  • Install: npm install -g @github/copilot
  • /model — select AI model
  • /init — setup instructions
  • @file — include files in context

Models we use: Claude Opus 4.6, GPT-5.3-Codex
